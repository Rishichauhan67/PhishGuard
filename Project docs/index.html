Phishing Detection System

AI / ML Working

This project uses Machine Learning to classify whether a URL is safe or phishing. The model is trained on a dataset of legitimate and malicious URLs.

1. Data Collection

Collected dataset containing phishing and safe URLs from online sources.

2. Feature Extraction

Important features are extracted such as URL length, presence of HTTPS, special characters, and domain age.

3. Model Training

Machine Learning algorithm like Random Forest is trained using the dataset.

4. Prediction

When a user enters a URL, the system extracts features and sends them to the trained model, which predicts whether the URL is phishing or safe.

5. Output

The result is displayed to the user as "Safe" or "Phishing".
